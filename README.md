# Go Tutorial Project

このリポジトリは、Goプログラミング言語の学習と練習のためのプロジェクトです。

## 構成

- `cmd/`: アプリケーションのエントリーポイントを含むディレクトリ
- `pkg/`: 外部アプリケーションで使用可能なライブラリコードを含むディレクトリ
- `internal/`: 内部パッケージを含むディレクトリ

## 実行方法

基本的なHello Worldプログラムを実行するには：

```bash
go run hello.go
```

## 依存関係の管理

このプロジェクトはGo Modulesを使用して依存関係を管理しています：

```bash
go mod tidy  # 依存関係を整理
```
